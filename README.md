# ⚙ Installation

### Clone the Repository

```bash
git clone https://github.com/RanbirProjects/Luxury-Printing.git
cd Luxury-Printing
```

---

### Install Dependencies

```
npm install
cd client
npm install
```

---

# ▶ Running the Application

### Start Backend

```
npm start
```

### Start Frontend

```
cd client
npm start
```

Open:

```
http://localhost:3000
```
